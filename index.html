PROJECTS


ROBO CAR

“Robo Car” is an IoT based device. It is built with various sensors and microprocessors along with car kit like wheels, chassis, wires, soldering kit and android phone. It runs by a code embedded in the microprocessor called Arduino UNO and is powered by an external source adapter which is than controlled by an android phone connected via Bluetooth with the car.

The purpose behind the project was to adapt a new technology and gain knowledge for the same. IoT has always been an emerging field in context of automation. So we tried to get hands-on on this diverse field and came out with an idea to make an automated car which can be controlled via phone.

SUNDAE SHOP STOCK MANAGEMENT SYSTEM

“Sundae Shop” is a Stock Management System completely developed on Java. It is a GUI based desktop application. It records stocks based on initial input given by the user at start of the application. Stocks can be managed as per the need and can also be printed if needed. The Sundae Shop contain 3 Windows: 1) Login Page for Admin, 2) Welcome Page and 3) Ordering Page.

The main aim to make this application was to store all the sundaes and print them all at the end of the day so that one would not found it tedious to account all the entries manually. Also inheriting only the linked pages not the content makes it invulnerable from any sort of fluctuation carried out by system.
